# ❓ Häufige Fragen (FAQ)

\<details\>

  \<summary\>Was ist dieses Portal?\</summary\>

  Dies ist die zentrale Dokumentation unserer API und Produkte.

  Hier findest du technische Details, Beispiele und Anleitungen.

\</details\>

\<details\>

  \<summary\>Was brauche ich, um die API zu nutzen?\</summary\>

  • Gültiger API-Key (vom Admin)  

  • Zugang zum Netzwerk oder zur Plattform

\</details\>

\<details\>

  \<summary\>Wo finde ich die OpenAPI-Spezifikation?\</summary\>

  Im **API Playground** kannst du die OpenAPI-Datei einsehen und herunterladen.

\</details\>

\<details\>

  \<summary\>Kann ich die API ohne Programmierkenntnisse testen?\</summary\>

  Ja. Im **Playground** Parameter eintragen und auf „Send“ klicken – die Antwort erscheint im Browser.

\</details\>

\</details\>
